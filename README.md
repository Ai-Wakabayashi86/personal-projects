# Merge Number

数字を育てて、つなげて、連鎖させるパズルゲームです。
限られたクリック数をやりくりしながら、より大きな数字を目指しましょう。

---

## ゲーム概要

* 7×7の盤面で遊ぶ数字パズルゲーム
* マスをクリックすると数字が1つ増加します
* 同じ数字が上下左右に3つ以上つながると自動で結合します
* 結合すると数字が1つ大きくなり、空いたマスは上から落下します
* 最上段には新しい数字がランダムで補充されます
* 連鎖を利用してクリック数を回復しながら高得点を目指します

---

## 遊び方

1. 好きなマスをクリックします。
2. クリックしたマスの数字が1増えます。
3. 同じ数字が3つ以上つながると自動で結合します。
4. 結合後は数字が1つ上がり、空いた場所へ上のマスが落下します。
5. 新しい数字が最上段から補充されます。
6. 連鎖が終わるまで結合を繰り返します。

---

## レベルアップ

ゲーム開始時は **1〜5** の数字が出現します。

残りクリック数が以下の値に到達するとレベルアップします。

| 残りクリック数 |  レベル |   出現数字 |
| ------: | ---: | -----: |
|      50 | Lv.2 |    1〜6 |
|     100 | Lv.3 |    1〜7 |
|     150 | Lv.4 |    1〜8 |
|     200 | Lv.5 |    1〜9 |
|     ... |  ... | 最大1〜15 |

レベルが上がるほど高い数字が出現し、より大きな連鎖を狙えるようになります。

---

## クリック数

クリック数はゲーム中の重要な資源です。

* マスを1回クリックすると **-1**
* 結合が1回発生すると **+1**

クリック数が **0を下回る** とゲームオーバーになります。

---

## 操作方法

| 操作     | 内容         |
| ------ | ---------- |
| クリック   | 数字を1増やす    |
| 遊び方ボタン | チュートリアルを表示 |
