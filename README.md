# 🌏 世界の国クイズ

世界193か国を対象にした学習用クイズゲームです。

## 機能

### 🇯🇵 国旗クイズ

国旗を見て国名を4択で答えます。

### 🍣 雑学クイズ

各国の特徴や文化、観光地などのヒントを見て国名を4択で答えます。

---

## 難易度

難易度は10段階に分かれています。

| レベル   | 名称   |
| ----- | ---- |
| Lv.1  | 初心者  |
| Lv.2  | 見習い  |
| Lv.3  | 冒険者  |
| Lv.4  | 熟練者  |
| Lv.5  | エリート |
| Lv.6  | 達人   |
| Lv.7  | 英雄   |
| Lv.8  | 伝説   |
| Lv.9  | 魔王軍  |
| Lv.10 | 魔王級  |

---

## 遊び方

1. クイズモードを選択

   * 国旗クイズ
   * 雑学クイズ

2. 難易度を選択

3. 10問連続で出題

4. 4択から正しい国名を選択

5. 結果発表

---

## 採点

* 全10問
* 正解数を表示
* 再挑戦可能

## 起動方法

```bash

```

ブラウザで開く

```text
http://localhost:8000
```

index.html をブラウザへドラッグ＆ドロップ
