# Personal Projects

このリポジトリは、趣味で作成したシステムやアプリを保存しているリポジトリです。

## 管理方法

各アプリ・システムは、ブランチごとに分けて管理しています。

- `main` ブランチ  
  リポジトリ全体の説明用ブランチです。

- `WorldQuiz` ブランチ  
  WorldQuiz アプリのソースコードを管理しています。

- `ThaiPrefecture` ブランチ  
  Thai Prefecture アプリのソースコードを管理しています。

## ブランチの確認方法

各プロジェクトの内容を確認する場合は、該当するブランチに切り替えてください。

```bash
git switch ブランチ名
